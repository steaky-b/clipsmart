# Clipsmart

A content creation tool built for social media creators.

## Live Site

Deployed via [Netlify](https://netlify.com) — auto-deploys on every push to `main`.

## Development

Plain HTML/CSS/JavaScript — no build step required.

Open `index.html` directly in a browser to preview locally.

## Deployment

Pushes to `main` trigger automatic Netlify redeploys.
